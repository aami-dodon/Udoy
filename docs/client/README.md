# Client Documentation

## Overview
React (Vite) application structured by features. Each feature mirrors a backend module and contains its own README.

## Routes
- `/` Landing marketing page.
- `/login` & `/signup` authentication flows.
- `/courses` catalog with course discovery.
- `/courses/:id` detail view with enroll action.
- `/dashboard/*` role dashboards.

## Testing
Run `npm run test -w client` to execute Vitest powered by React Testing Library.
