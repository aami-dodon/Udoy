## As a
User

## I want to
control visibility of my profile fields

## Acceptance Criteria
- Per-field visibility options available
- Defaults follow child-safety policies
- UI reflects visibility choices immediately
- Changes audited and reversible

- Tests passed

- Deployed to staging
